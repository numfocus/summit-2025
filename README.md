# summit-2025

This repository contains the assets (recordings, notes and presenatation slides) from the 2025 NumFOCUS Project Summit - held October 22-24.
The summit was held online via Zoom

Attendees:
- Sponsored project communities
- Affiliated project communities
- NumFOCUS Admin Board
- NumFOCUS Technical Steering Committee
- NumFOCUS Advisory Council
- NumFOCUS Volunteer Committee members

Please reach out to [help@numfocus](help@numfocus.org}) if you have any questions or issues with access.

Enjoy!
